## CSS Gerador com IA ⚡️

Um projeto simples que gera código HTML e CSS usando inteligência artificial. Você digita o que imagina, e a IA transforma em código pronto, exibindo o resultado na tela.

## Como usar

1. Abra o projeto.


2. Digite sua ideia no campo de texto.


3. Clique em "Gerar Código ⚡️".


4. Veja o código e o preview funcionando em tempo real.
